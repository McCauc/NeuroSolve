# Technology Stack

This document outlines the technology stack for the NeuroSolve project.

## Core Language
*   **Python:** The project is built entirely in Python.

## User Interface
*   **Streamlit:** The user interface is a web application powered by the Streamlit framework.

## Numerical and Data Handling
*   **NumPy:** Used for core numerical operations and array manipulation.
*   **Pandas:** Used for data structures and analysis, particularly for handling and exporting results.

## Symbolic Mathematics
*   **SymPy:** Used for all symbolic math operations, including exact derivatives and integrals.

## Visualization
*   **Plotly:** The primary library for creating interactive plots and visualizations.
*   **Matplotlib:** Used for generating static plots or as a fallback for specific visualization needs.
